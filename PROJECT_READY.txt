🎯 QUICK BUILD CHECKLIST (To Get Running)
───────────────────────────────────────────────────────────────────

☐ STEP 1: Setup MongoDB
  - Go to https://www.mongodb.com/cloud/atlas
  - Create FREE cluster
  - Get connection string
  - Takes ~5 minutes

☐ STEP 2: Get Bolna Credentials
  - Go to https://platform.bolna.ai
  - Get API Key from Developers section
  - Create an Agent and get Agent ID
  - Takes ~5 minutes

☐ STEP 3: Update .env File
  - Edit .env in this folder
  - Add MONGODB_URI (from Step 1)
  - Add BOLNA_API_KEY (from Step 2)
  - Takes ~1 minute

☐ STEP 4: Start Server
  - Run: npm run dev
  - Wait for "✅ MongoDB connected"
  - Takes ~30 seconds

☐ STEP 5: Open Dashboard
  - Go to: http://localhost:5000
  - Start using the app!
  - Takes ~1 minute

TOTAL TIME: ~15 minutes ⏱️

🔌 API ENDPOINTS (All Ready to Use)
───────────────────────────────────────────────────────────────────

Customer Management:
  GET    /api/customers               → List all customers
  POST   /api/customers               → Create new customer
  GET    /api/customers/:id           → Get customer details
  PUT    /api/customers/:id           → Update customer
  DELETE /api/customers/:id           → Delete customer

Call Management:
  GET    /api/calls                   → List all calls
  GET    /api/calls/:id               → Get call details
  POST   /api/calls/trigger           → 🔴 TRIGGER OUTBOUND CALL
  GET    /api/calls/:id/status        → Check call status
  GET    /api/calls/:id/transcript    → Get conversation transcript
  GET    /api/calls/:id/recording     → Get call recording
  POST   /api/calls/:id/transfer      → Transfer to human agent
  GET    /api/calls/stats/summary     → Get analytics

System:
  GET    /api/health                  → Server health check
  POST   /webhooks/bolna              → Bolna callback webhook
  POST   /webhooks/test               → Test webhook
