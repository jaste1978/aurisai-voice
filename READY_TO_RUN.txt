🎯 YOUR NEXT 3 STEPS (To Get Running In 15 Minutes)
═══════════════════════════════════════════════════════════════════

STEP 1: Setup MongoDB (5 minutes)
├─ Go to: https://www.mongodb.com/cloud/atlas
├─ Sign up FREE
├─ Create cluster (M0 tier)
├─ Create user (username: bolna)
├─ Get connection string
└─ Copy to clipboard (save for Step 3)

STEP 2: Get Bolna Credentials (5 minutes)
├─ Go to: https://platform.bolna.ai
├─ Get API Key from Developers
├─ Create an Agent
├─ Copy Agent ID
└─ Save both for Step 3

STEP 3: Configure & Start (5 minutes)
├─ Edit .env file:
│  MONGODB_URI=<paste from Step 1>
│  BOLNA_API_KEY=<paste from Step 2>
│
├─ Start server:
│  npm run dev
│
├─ Wait for:
│  ✅ MongoDB connected successfully
│  🚀 Server running on port 5000
│
└─ Open Dashboard:
   http://localhost:5000

📞 ONCE IT'S RUNNING - MAKE YOUR FIRST CALL
═══════════════════════════════════════════════════════════════════

1. Go to Customers tab
   └─ Click "Add Customer"
   └─ Fill in: name, phone (+1234567890), language
   └─ Click "Add Customer"

2. Go to https://platform.bolna.ai/agents
   └─ Create an agent
   └─ Copy the Agent ID

3. Go to "Trigger Call" tab
   └─ Select the customer from dropdown
   └─ Paste Agent ID
   └─ Click "Trigger Call"

4. Go to "Calls" tab
   └─ View call status
   └─ Once complete, click "View"
   └─ See transcript and recording
