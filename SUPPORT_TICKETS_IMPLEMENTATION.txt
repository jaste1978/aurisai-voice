💡 HOW IT WORKS
─────────────────────────────────────────────────────────────────────────────

1. AI Call Completes
   ↓
2. Customer provides email address
   ↓
3. AI extracts: support_email = "customer@example.com"
   ↓
4. Bolna sends webhook with agent_extraction
   ↓
5. Our server creates Freshdesk ticket
   ↓
6. Ticket info saved to database
   ↓
7. Support Tickets page queries and displays it

🔌 API ENDPOINT
─────────────────────────────────────────────────────────────────────────────

GET /api/calls/support-tickets

Query Parameters:
  page=1          (default: 1)
  limit=20        (default: 20, options: 10,20,50,100)
  agentId=xxx     (optional: filter by agent)
  status=yyy      (optional: filter by call status)

Example:
  GET /api/calls/support-tickets?page=1&limit=20&agentId=a7487186&status=completed

Response:
  {
    "success": true,
    "data": [ /* array of tickets */ ],
    "pagination": {
      "currentPage": 1,
      "totalPages": 3,
      "totalRecords": 45
    }
  }
